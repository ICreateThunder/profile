# Code of Conduct

Participation in this project - the repository, issues, pull requests, discussions, and any other connected communication channel - is governed by this minimum:

- **Be respectful** in disagreement. Critique work, not people.
- **Assume good faith.** Most contributors are volunteers doing their best.
- **No harassment, personal attacks, or discrimination.**
- **No sharing of private communications** without the other party's consent.
- **Acknowledge the time of others.** Maintainers, reviewers, and reporters are not on retainer.

Disputes are resolved by the maintainers. Reports of unacceptable behaviour go to `robert@shalders.co.uk`, or via [GitHub Private Security Advisories](https://github.com/ICreateThunder/profile/security/advisories/new) if a private channel is preferred. Maintainers may warn, suspend, or ban anyone whose behaviour materially disrupts the project; persistent or serious violations result in a permanent ban.

This document is deliberately short. A longer code is not a stronger one - we'd rather contributors read it than skim it.
