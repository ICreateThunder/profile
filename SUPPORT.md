# Support

How to get help, and which channel to use.

| You want to… | Use |
|---|---|
| Ask an open-ended question | [GitHub Discussions](https://github.com/ICreateThunder/profile/discussions) |
| Report a reproducible bug | [Bug report issue](https://github.com/ICreateThunder/profile/issues/new/choose) |
| Request a feature | [Feature request issue](https://github.com/ICreateThunder/profile/issues/new/choose) |
| Report a security vulnerability | **Privately** - see [SECURITY.md](SECURITY.md). Do **not** open a public issue. |
| Understand how to contribute | [CONTRIBUTING.md](CONTRIBUTING.md) |

This is a personal project maintained in spare time. Issues and discussions are
read, but there is no support SLA except for security reports, which follow the
response targets in [SECURITY.md](SECURITY.md).
